This project is a simple weather web application built with **vanilla JavaScript**. It uses the **Visual Crossing Weather API** to fetch real-time weather data and display it on the page.

The main goal of this project was to practice:
- Working with APIs
- Getting familiar with promises and `async/await`
- Linting and formatting code with ESLint and Prettier
- Getting more familiar with using Webpack
- Organizing JavaScript code

This project is part of my learning journey through **The Odin Project**.

## Features

- Search weather by city/town name  
- Displays current temperature and weather conditions  
- Displays weather icons depending on weather conditions
- Fetches real data from an external API  
- Handles asynchronous JavaScript using `async/await`
- Enforces consistent code style with ESLint and Prettier

## Getting Started

### Prerequisites

- Node.js
- npm

---

### Installation and running locally

```bash
git clone https://github.com/ikonyx/weather-app.git
cd weather-app
npm install
npm run dev
```

Then visit [http://localhost:8080](http://localhost:8080)
